# Data

The data used in this project comes from the OpenBiomechanics baseball pitching dataset provided by Driveline Baseball.

Due to file size limitations, the raw data is not included in this repository.

## Download Instructions

Download the dataset from:
[https://github.com/drivelineresearch/openbiomechanics/tree/main/baseball_pitching/data/full_sig]

After downloading, place the following CSV files in this folder:

- metadata.csv  
- joint_velos.csv  
- force_plate.csv  
- energy_flow.csv  

## Notes

- This repository only contains analysis code and derived results  
- The dataset is publicly available and not redistributed here
